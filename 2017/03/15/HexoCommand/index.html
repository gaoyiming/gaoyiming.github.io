Hexo常用快捷键

发表于 2017-03-15   |   分类于 博客   |     |   次阅读

常用命令

init

新建一个网站。如果没有设置 folder ，Hexo 默认在目前的文件夹建立网站。

new

新建一篇文章。如果没有设置 layout 的话，默认使用 _config.yml 中的 default_layout 参数代替。如果标题包含空格的话，请使用引号括起来。

例如 本文我生成命令是

会生成一个HexoCommand.md文档，文档里边有一个title关键字，那个决定的你文章显示什么标题，这个HexoCommand只是会在你发布之后存在在你的url里边，来方便url定向。
我的url：http://gaoyiming.site/2017/03/15/HexoCommand/

所以说这个并不是你文章的title（虽然默认是），他会决定你url，所以尽量写成英文（关键就是这一句）。

generate

生成静态文件

publish

生成草稿

server

启动服务器，之后可以在 http://localhost:4000/ 进行查看

deploy

部署网站。例如推到github
这个需要在站点配置文件重配置一下
例如我的配置，大家对应配置一下就好

clean

清除缓存文件 (db.json) 和已生成的静态文件 (public)。

在某些情况（尤其是更换主题后），如果发现您对站点的更改无论如何也不生效，您可能需要运行该命令。

基本上本人常使用的就这些，还有一些快捷键没有经常使用，就不提了。
