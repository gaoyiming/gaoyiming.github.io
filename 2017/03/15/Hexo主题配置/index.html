GitHub Pages 绑定域名

创建CNAME文件夹
在你的个人博客仓库的根目录中新建文件CNAME（注意没有后缀,而且一定要大写大写大写），在该文件增加一行文字，告诉Github Pages服务器你想指定的域名。该域名不能包含前缀信息，即不能添加http:\前缀。

gaoyiming.site

然后上传到github
CNAME绑定域名
进入你的域名控制台，添加域名解析

记录类型 主机记录 解析线路 记录值
CNAME @ 默认 your.github.io

然后保存即可
